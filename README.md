### Instruções para rodar:
- Instalar as dependências:
```bash
npm install
```
- Rodar o projeto:
```bash
npm start
```
- Ou para rodar em modo de desenvolvimento:
```bash
npm run dev
```

### Versão utilizada do Node.js: 10.7.0
